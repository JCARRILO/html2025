Clase 1 — Fundamentos HTML

Introducción

Archivo base con doctype, elementos html, head y body. Esta guía unifica todas las lecciones sin duplicar código.

Estructura básica

<!doctype html>
<html lang="es">
  <head> ... </head>
  <body> ... </body>
</html>

Ejemplo rápido

Agrega tu contenido dentro de main siguiendo la jerarquía de títulos. Usa utilidades comunes para accesibilidad y layout.
