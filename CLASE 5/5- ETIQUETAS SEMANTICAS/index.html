ETIQUETAS SEMÁNTICAS: ARTICLE vs SECTION

1. Cuando el ARTICLE contiene SECTIONS

Se usa cuando el contenido puede existir de forma independiente. Este artículo podría publicarse solo, por ejemplo como una entrada de blog o una guía. Dentro se incluyen varias secciones para organizar sus partes.

Parte A — Concepto general

Un article representa una unidad completa: algo que tiene sentido por sí mismo. Dentro de él se pueden incluir secciones que desarrollen distintos aspectos o capítulos del mismo tema.

Parte B — Cuándo conviene

  • Entradas de blog
  • Noticias o comunicados
  • Fichas de producto o guías individuales

2. Cuando una SECTION agrupa varios ARTICLES

Se usa cuando se desea presentar varios artículos relacionados entre sí, como un conjunto de noticias o publicaciones. La sección actúa como contenedor temático y cada artículo mantiene su independencia.
