Cómo estructurar HTML y preparar meta etiquetas para SEO

19 octubre 2025 · Equipo del curso

Etiquetas semánticas básicas

  • Landmarks: header, nav, main, aside, footer.
  • Contenido: article, section, h1–h6, figure/figcaption.
  • Textos alternativos y jerarquía de encabezados coherente.

Meta etiquetas esenciales para SEO

  • <title> conciso (50–60 caracteres).
  • meta name="description" (≈ 155 caracteres).
  • <link rel="canonical"> para URL preferida.
  • Open Graph y Twitter Card para compartir en redes.
  • meta name="robots": index,follow o noindex,nofollow según corresponda.

Schema.org con JSON‑LD

Incluye datos estructurados para enriquecer resultados: Organization, WebSite + SearchAction, BreadcrumbList y Article.

Preguntas frecuentes

¿Cuántos meta debo usar?

Al menos title, description, canonical y robots. Añade Open Graph/Twitter para redes y JSON‑LD para datos estructurados.

¿Es obligatorio usar JSON‑LD?

No es obligatorio, pero ayuda a conseguir resultados enriquecidos (breadcrumbs, FAQ, artículos, etc.).
