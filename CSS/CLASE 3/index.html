CSS - Clase 3: Pseudoclases y Pseudoelementos

Estados con :focus, selección estructural con :first-child/:last-child/:nth-child y contenido generado con ::before/::after.

¿Cómo estilizar estados con pseudoclases de CSS?

Las pseudoclases permiten apuntar a estados del DOM sin añadir clases. Un caso típico es :focus en un input: al recibir foco por clic o teclado, podemos cambiar su estilo.

:focus — ¿Qué hace y cómo aplicarlo?

  • :focus estiliza el elemento enfocado.
  • Útil para guías visuales y accesibilidad.
  • Combina outline y background-color para feedback inmediato.
input:focus {
  outline: 2px solid #3498DB;
  background-color: #E8F4F8;
}

¿Cómo seleccionar elementos específicos en listas?

:first-child y :last-child apuntan al primer y último hijo, evitando clases extra en cada elemento.

  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemento 4
li:first-child {
  color: green;
  font-weight: bold;
}

li:last-child {
  color: red;
  font-weight: bold;
}

¿Para qué sirve :nth-child(even) en tablas?

Alterna estilos por posición (pares/impares). Escala con muchas filas sin añadir clases manuales.

¿Cómo crear contenido con pseudoelementos?

Con ::before y ::after puedes generar contenido sin modificar el HTML usando content.

El CSS puede generar comillas u otros detalles visuales.

.quotes::before {
  content: '"';
  color: blue;
  font-size: 24px;
}

.quotes::after {
  content: '"';
  color: blue;
  font-size: 24px;
}
