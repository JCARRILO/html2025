1) Sintaxis de CSS

Una regla CSS tiene selector y declaraciones (propiedad: valor) dentro de llaves.

/* Comentario */
selector {
  propiedad: valor;
  otra-propiedad: otro-valor;
}

/* Ejemplo */
.btn {
  background: #2f6e93;
  color: #fff;
}
  • Los puntos y comas separan declaraciones.
  • Los nombres de propiedades son en minúsculas y con guiones.
  • Evita !important salvo excepciones; aprende especificidad.
